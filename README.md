## About Me

Full-stack developer focused on building fast, clean, and production-ready SaaS products, MVPs, AI-powered apps, and mobile applications.

I work with React, React Native, Next.js, Python, Supabase, PostgreSQL, and cloud platforms to turn product ideas into real working applications.

My focus is simple. Move fast, keep the structure clean, and build products that real users can actually use.

I also use modern AI development tools like Claude, Cursor, OpenAI, Lovable, and Base44 to speed up development while keeping the code, product logic, and architecture under control.
